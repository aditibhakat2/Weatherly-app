# 🌙 Weatherly

A modern, responsive weather application that provides real-time weather updates using WeatherAPI.

## 🚀 Features

* 🌍 Search weather by city name
* 🌡️ Real-time temperature display
* ☁️ Weather condition + icons
* 🎨 Dynamic background based on weather
* 🌙 Dark mode UI with glassmorphism
* ⌨️ Enter key support

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* WeatherAPI
